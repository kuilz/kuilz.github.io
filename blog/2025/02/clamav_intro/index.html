Ubuntu杀毒软件ClamAV使用说明

ClamAV is an open-source antivirus engine for detecting trojans, viruses, malware & other malicious threats.

# 安装
sudo apt install clamav

# 使用
sudo clamscan -r -v / --move=/virus_file  -l detect.log

常用参数说明：

  • -r: 递归扫描指定目录及其子目录中的所有文件。
  • -i: 只显示被感染的文件。
  • -v: 显示详细的扫描信息，包括每个被扫描的文件。
  • -l <file>: 将扫描结果输出到指定的日志文件。
  • --remove: 删除发现的病毒文件。注意，使用此选项会直接删除病毒文件，不会将其隔离。
  • --move=<dir>: 将感染的文件移动到指定的目录，而不是删除它们。需要提供目标目录路径。

不建议使用--remove直接删除，建议使用--move进行隔离，防止误删重要文件。
